科技茶坛002：珠联璧合！Visual Studio Code的安装与配置

前言


千里之行，始于足下。

无论一个程序员的编程水平有多么高超，

他的编程学习都从他的第一行代码开始。

相比于上世纪的八九十年代的编程学习者，

不得不说现在的编程学习者实在是太幸福了！

当今编程基础教育层林尽染，编程教学参考百花齐放，

也难怪名家都说：全民编程的时代已然来临了！

简介


Visual Studio Code，简称VSC，

是一款由微软公司开发并跨平台的免费源代码编辑器。

该软件支持语法高亮、代码自动补全、查看定义等功能，

除此之外甚至其还内置了命令行工具和Git版本控制系统，

同时，用户可以更改主题和键盘快捷方式实现个性化设置，

也可通过内置的扩展程序商店安装扩展以拓展软件功能。

实在是单人单撸，多人运动，阖家删库跑路的必备软件！


所以，别苦着个脸了，现在来开始我们的安装吧！

教程


首先在安装前，你需要冷静下来思考这样一个问题：

VSC跟学校推荐的VS、DEV、CB这些软件的区别是什么？


显然，你会发现VSC是一个代码编辑器，而VS等是集成式开发环境…

所谓集成开发环境，是一种辅助程序开发人员开发软件的应用软件，

它可以辅助你编写源代码文本，并且帮你编译打包生成可用的程序。

而一般来说，集成开发环境大都会包含有代码编辑器以及编译器，

所以当你下载了VSC之后，你还需要配置好你的编译器，猛男落泪！


但是阁下，和网上那群虫豸配置教程在一起，怎么能安装好VSC啊？

所以我自己搞了个整合包，它可以帮助你快速配置好C/C++的开发环境！

你只需要直接下载，然后就根据整合包内的教程安装VSC并配置开发环境了！

下载链接：https://share.weiyun.com/HRdnVSTx


获得整合包后，可将其其解压至任意位置，

之后请右键点击install.bat文件使其运行，

若运行无误，敲击回车即可退出。

但是，请事先移除Program File中的mingw64，

否则就会出现这样的离谱提示！

之后点击VSCodeSetup.exe文件，

同意协议，选择默认路径，

之后将Other项全部勾选！

完成安装后启动VSC，

点击左侧Extensions,

搜索并安装以下插件：

C/C++

Code Runner

Chinese (Simplified) Language Pack

然后点击右下显示的”Yes”，重新加载即可实现切换语言。

最后请在纯英语路径下创建将要编写c/cpp的文件夹（如 c:\projects），

否则，文件在运行或者进行debug的时候会报错，极其影响体验，

并将整合包中Demo对应的c/cpp里的.vscode文件夹复制到此处，

创建用于编写语言的文件夹（如 projects）内，


最后右键该文件夹（如 projects），Open with Code，

即可开始你的编程!
